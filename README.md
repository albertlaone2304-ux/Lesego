# Claws by Lesh — Website

## Folder Structure

```
claws-by-lesh/
│
├── index.html        (Home Page)
├── about.html        (About Page)
├── services.html     (Services & Pricing Page)
├── gallery.html      (Gallery Page)
├── booking.html      (Booking Page)
├── contact.html      (Contact Page)
├── feedback.html     (Feedback Page)
├── style.css         (External Stylesheet — used by all pages)
│
├── photos/           (Create this folder and add your 6 photos)
│   ├── photo1.jpg
│   ├── photo2.jpg
│   ├── photo3.jpg
│   ├── photo4.jpg
│   ├── photo5.jpg
│   └── photo6.jpg
│
└── videos/           (Create this folder and add your 3 videos)
    ├── video1.mp4
    ├── video2.mp4
    └── video3.mp4
```

## How to Add Your Photos & Videos

1. Create a folder called `photos` inside the `claws-by-lesh` folder.
2. Add your 6 photos and name them: photo1.jpg, photo2.jpg ... photo6.jpg
3. Create a folder called `videos` inside the `claws-by-lesh` folder.
4. Add your 3 videos and name them: video1.mp4, video2.mp4, video3.mp4
